# 🏦 Bank Churn Analysis Project

## 📌 Overview

This project focuses on analyzing customer churn in a bank using data analysis techniques. The goal is to identify key factors that influence customer churn and provide actionable insights.

---

## 📂 Dataset

* Customer data including demographics, account details, and activity
* Features include:

  * Geography
  * Gender
  * Age
  * Tenure
  * Balance
  * Number of Products
  * Credit Card Status
  * Active Member Status
  * Estimated Salary

---

## 🛠️ Tools & Technologies Used

* Python (Pandas, NumPy, Matplotlib, Seaborn)
* SQL (for data querying and analysis)
* Jupyter Notebook
* VS Code

---

## 📊 Key Analysis Performed

* Data Cleaning & Preprocessing
* Exploratory Data Analysis (EDA)
* Customer segmentation
* Churn pattern identification

---

## 🔍 Key Insights

* Customers with lower activity are more likely to churn
* Balance and number of products impact churn behavior
* Certain geographic regions show higher churn rates

---

## 🗂️ Project Structure

* `churn_analysis.ipynb` → Main analysis notebook
* `bank_churn.sql` → SQL queries used in analysis
* `bank_churn_cleaned.csv` → Cleaned dataset
* `README.md` → Project documentation

---

## 🚀 How to Run

1. Clone the repository
2. Open the notebook in Jupyter or VS Code
3. Run all cells to reproduce analysis
